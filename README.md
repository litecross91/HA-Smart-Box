# HA-Smart-Box
Home Assistant ESP32-S3 Smart Box Project

- Hőmérséklet/páratartalom: DHT22
- Hangszóró: 3W 4Ω
- Amp: MAX98357A
- Kijelző: 1,3" OLED I2C
- Nyomógomb: 5 db
- LED visszajelző: 5 db
- Nyomógombos rotary enkóder modul: 1 db
- Box: 115x90x55 cm
